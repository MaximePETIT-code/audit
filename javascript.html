Cheat code

En fouillant les fichiers sources et plus précisément le fichier "main.js" je suis tombé sur un "cheat code" menant à un dialogue d'alerte.

Évènement "dblclick"

Statut : critique

Problème : Dans "cookies-modal.js", l'évènement "dblclick" est utilisé pour envoyé le formulaire présent sur la fenêtre modale de gestion des cookies. L'évènement "dblclick" est activé lorsque le clique gauche de la souris est cliqué à deux reprises dans un temps très court (soit après deux évènements "click"). Dans ce cas la il n'est pas pertinent d'utiliser dblclick puisque cela rend l'envoi du formulaire compliqué.

Recommandation : Dans le cadre d'une démarche visant à améliorer le code javascript, nous vous recommandons l'axe d'amélioration suivant :

  • Utitliser l'évènement click
